# GitHub User Search Application

A React application for searching GitHub users using the GitHub API.

## Live Demo
[View Live App](https://alx-fe-reactjs-fuw1.vercel.app/)

## Features
- Search GitHub users by username
- Filter by location and minimum repositories
- View user profiles with avatar and details
- Load more results with pagination

## Technologies
- React + Vite
- Axios
- Tailwind CSS
